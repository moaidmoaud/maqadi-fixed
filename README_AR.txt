مشروع تطبيق مقاضي للأندرويد

هذا المشروع يحول واجهة maqadi_v2.html إلى تطبيق أندرويد يعمل دون إنترنت باستخدام WebView.

طريقة استخراج APK من المتصفح فقط عبر GitHub:
1) افتح github.com وسجل الدخول.
2) أنشئ مستودعًا جديدًا باسم maqadi-android.
3) ارفع جميع محتويات هذا المجلد، بما فيها مجلد .github.
4) افتح تبويب Actions داخل المستودع.
5) اختر Build Android APK ثم Run workflow.
6) بعد اكتمال البناء افتح التشغيل الأخير، وانزل إلى Artifacts.
7) حمّل Maqadi-APK، وفك الضغط؛ ستجد app-debug.apk.
8) أرسل APK إلى جوال أندرويد وافتحه للتثبيت.

ملاحظة: نسخة debug صالحة للتثبيت والتجربة الشخصية. النشر على Google Play يحتاج نسخة Release موقعة بمفتاح خاص.
